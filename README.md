# 💬 Chaty - Ephemeral Web-Based Chat Platform

Chaty is a secure, temporary chat room generator designed with a **privacy-first** approach. It enables instant communication with a **zero-data-retention** architecture, ensuring that your conversations remain private and are cleared as soon as the session ends.

---

## ✨ Key Features

- 🚀 **Instant Room Generation**: Create a unique chat room with a single click.
- 🛡️ **Zero-Data Retention**: Messages are stored temporarily and cleared automatically.
- 🎨 **Modern UI/UX**: Aesthetic design with glassmorphism elements and full responsiveness.
- 💬 **Real-time Interaction**: Simulated real-time messaging with typing indicators.
- 🔗 **Easy Sharing**: Built-in sharing options for WhatsApp, Telegram, and Email.
- 🗄️ **No Database Required**: Lightweight implementation using file-based JSON storage.

---

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom styles), JavaScript (ES6+)
- **Backend**: PHP (Session management and File I/O)
- **Data Storage**: Local JSON-based file system (No SQL/NoSQL database needed)
- **Communication**: AJAX Polling for simulated real-time updates

---

## 🚀 Getting Started

### Prerequisites
- A web server with **PHP 7.4+** support (e.g., Apache, Nginx, or XAMPP/WAMP for local development).

### Setup
1. Clone the repository or download the source code.
2. Place the files in your server's web root (e.g., `htdocs` or `www`).
3. Ensure the server has **write permissions** for the directory to create the `chats/` folder.
4. Open `index.html` in your browser.
